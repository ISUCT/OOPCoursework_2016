Предметная область — разработка компьютерной игры кликер. Необходимо разработать компьютерную игру, которая может:
Запускать игровую сессию, где пользователю предлогается нажимать на кнопку.
Хранить лучший результат игрока после завершения игровой сессии.
Проигрывать звуковой сигнал при нажатии на кнопку.
