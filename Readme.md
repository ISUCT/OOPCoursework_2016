Техническое задание.

 Создать внутри-терминальную программу позволяющую :

 1. Выбирать готовый текстовый файл или вводить/редактровать текст.
 2. Осуществлять передачу текста на сервис yandex, для генерации аудиофайла на основе текста.
 3. Получать и обрабатывать ответ от сервиса yandex.
 4. Воспроизведить полученный аудиофайл.
